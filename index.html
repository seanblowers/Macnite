Macnite

Install your favorite Mac apps in one go.

Tick the apps you want. We'll build a single command that installs them all — no setup wizards, no clicking through installers, no signups.

Free. Open source. No accounts. No tracking.

How it works

  1. 1
    Tick the apps you want

    Pick from the popular list, or search for anything else.

  2. 2
    Copy the command

    One button copies a single install command to your clipboard.

  3. 3
    Paste it into Terminal

    Open Terminal (press ⌘+Space, type "Terminal"), paste, and press Enter. Done.

Macnite uses Homebrew, the standard package manager for Mac. If you don't have it yet, the downloadable installer script sets it up for you.
